Garmin Chat Connector

Connect your Garmin account to Claude, ChatGPT, and other AI chat tools.

Browser-based authentication required. Run the one-time setup script on your computer to connect your Garmin account. The script opens a real Chrome window so Garmin's security checks pass correctly.

How to connect your account:

  1. Install the setup script dependencies (Python 3.10+ required):
    pip install playwright httpx playwright install chrome
  2. Get the setup script — find scripts/playwright_setup.py in the project repository and copy it to your machine.
  3. Run the script — it will open a Chrome window for login:
    python playwright_setup.py
    Enter your Garmin email, password, and this server's URL when prompted. Complete any MFA challenge in the browser window.
  4. Copy your MCP URL from the script output and add it to your Claude Desktop config or any MCP-compatible client.
Why a local script? Garmin's login page uses Cloudflare Turnstile, which requires a real browser on your machine to pass verification. The script authenticates once, saves a session that lasts ~1 year, and uploads only the encrypted session token (never your password) to this server. Re-run it when your session expires.
